📔 Digital Diary Web Application
A secure and user-friendly Digital Diary Web App that allows users to write, manage, and store personal notes online with authentication and real-time updates.
🚀 Features
🔐 User Authentication (Signup/Login) using Firebase
📝 Create, Read, Update, Delete (CRUD) diary entries
⚡ Real-time data synchronization
📱 Responsive and clean UI
☁️ Cloud-based storage for secure access
🔎 Easy navigation and user-friendly interface
